 # LumE USD (LUSD)

**LumE USD (LUSD)** — это децентрализованный стейблкоин, выпущенный на блокчейне Solana, созданный для надёжных, быстрых и дешёвых транзакций. Подходит для DeFi, Web3-приложений и повседневных платежей.
